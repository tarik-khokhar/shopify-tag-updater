01 Shopify Connection
⚠
Browser CORS limitation
Shopify's Admin API blocks direct browser requests. To use this app fully, either:
Option A — run a local proxy: npx shopify-proxy --token YOUR_TOKEN --store YOUR_STORE
Option B — deploy a Vercel serverless function (see previous guide).
Enter your proxy URL above, or use the Dry Run toggle below to preview changes without writing.
